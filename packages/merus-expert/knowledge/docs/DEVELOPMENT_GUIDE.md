# Development Guide - Merus Expert

MerusCase matter automation framework development guide.

## Overview

Merus Expert is a SOC2-compliant browser automation tool for creating matters in MerusCase using Playwright and Browserless API.

## Setup

### Prerequisites

- Python 3.12+
- Browserless API access
- MerusCase credentials

### Installation

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install chromium

# Initialize database
python main.py init-db
```

## Key Components

### Browser Client

```python
from browser.client import BrowserlessClient

async with BrowserlessClient() as client:
    await client.navigate("https://meruscase.com")
    await client.fill_form(matter_data)
```

### Element Handler

Uses multiple strategies for element finding:
1. ID selector
2. Name attribute
3. Label association
4. XPath fallback

### Knowledge Database

SQLite database storing:
- Court information
- Judge data
- Case type mappings
- Attorney lookups

## Security Considerations

- SOC2 compliance required
- Credentials stored securely (environment variables)
- Audit logging enabled
- Session isolation

## Troubleshooting

### Connection Issues
```bash
# Test Browserless connectivity
python main.py test-browser
```

### Database Errors
```bash
# Reinitialize database
rm knowledge/db/merus_knowledge.db
python main.py init-db
```
