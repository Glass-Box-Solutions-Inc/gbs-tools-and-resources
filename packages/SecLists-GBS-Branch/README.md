## What It Is

SecLists is the security tester's companion — a collection of multiple types of lists used during security assessments: usernames, passwords, URLs, sensitive data patterns, fuzzing payloads, web shells, and more. This is a GBS branch/fork of that collection used for internal security testing reference.
